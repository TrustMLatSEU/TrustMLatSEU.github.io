Kimi API Key 获取指南

1

访问Kimi开发者平台

首先需要登录Kimi的开发者平台获取API Key：

前往Kimi开发者平台
如果您已有Kimi账号： 直接使用账号登录
如果没有账号： 点击注册并完成账号创建
2

导航至API Key管理页面

登录成功后，按照以下步骤找到API Key管理页面：

在左侧导航栏中找到并点击 "API Keys" 或 "API管理"
提示： 如果您是第一次使用，可能需要先同意开发者协议或完成身份验证。
3

创建新的API Key

在API Key管理页面中：

点击 "创建新的API Key" 按钮
根据需要填写API Key的名称（可选）
确认创建，系统将生成一个唯一的API Key
重要： API Key创建后请立即复制保存，关闭页面后将无法再次查看完整Key。
4

复制并保存API Key

成功创建API Key后：

立即复制 生成的API Key（点击复制按钮或手动选择复制）
将复制的API Key 安全保存 在您信任的地方
建议： 使用密码管理器保存API Key，避免使用明文存储在不安全的地方。
5

在平台中使用API Key

现在您可以将API Key用于我们的平台：

返回平台，在 "Kimi API Key" 输入框中粘贴您的Key
点击保存或确认，即可开始使用Kimi API服务
测试： 保存后可以尝试进行一次查询，确认API Key工作正常。
返回平台

常见问题

API Key泄露了怎么办？
立即返回Kimi开发者平台，删除泄露的Key并创建新的Key。
API Key无效？
检查是否复制了完整Key，确保没有多余空格。如果问题持续，尝试创建新Key。
如何保证API Key安全？
不要将Key提交到公开代码库，使用环境变量存储，定期轮换Key。
